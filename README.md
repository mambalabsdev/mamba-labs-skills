# Mamba Labs Skills

Free skills that teach AI agents to run signal-based GTM workflows. Works with Claude Code, Codex CLI, Cursor, and other MCP-compatible agents.

## Skills
- [ICP Definition Builder](skills/icp-definition-builder) — build a structured ICP from a business description.
- [Cold Email Structure Checker](skills/cold-email-structure-checker) — audit a cold email draft against a 7-point framework.
- [Domain Health Pre-Check](skills/domain-health-precheck) — check SPF, DKIM, DMARC, and blacklist status for a sending domain.
- [Hiring Signal Spotter](skills/hiring-signal-spotter) — analyze a company's GTM hiring activity for buying signals.

## Install
Drop any skill folder into `~/.claude/skills/` to use it in Claude Code.
